## Requirements

```sh
paru -S neovim-nightly-bin carapace-bin kitty stow cmake ttf-jetbrains-mono-nerd git-delta
cargo install zoxide starship nu
```

## Usage

```sh
git clone git@github.com:jerzywilczek/dotfiles.git ~/dotfiles
cd ~/dotfiles
stow --dotfiles .
```
